
# Pastebin Lite (MERN + Next.js)

## Description
A lightweight Pastebin-like app with TTL and view limits.

## Tech Stack
- MongoDB, Express, Node.js
- Next.js (App Router)

## Run Locally
Backend:
- cd backend
- npm install
- node server.js

Frontend:
- cd frontend
- npm install
- npm run dev

## Persistence
MongoDB
